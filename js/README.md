# dev/js 모듈 구성

- `config.js`: Firebase/상수/센터/공휴일/버전
- `utils.js`: 공통 유틸
- `reservation-data.js`: Firestore 예약 조회/캐시
- `auth.js`: 로그인/회원/소셜로그인
- `calendar.js`: 달력/날씨
- `reservation-ui.js`: 예약판 UI/코트 선택/공유
- `booking-helpers.js`: 예약 검증/금액/매크로 보조
- `booking-core.js`: 예약 모달/결제/doPay/saveBook
- `admin-actions.js`: 관리자 예약/승인/취소/정기예약 처리
- `admin-settings.js`: 관리자 설정/통계/시민인증 관리
- `my-reservations.js`: 내예약/더치페이/전적관리
- `board.js`: 게시판/댓글/취소요청 게시글
